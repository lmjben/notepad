## 安装

``` shell
git clone https://github.com/yhlben/notepad.git
cd notepad
yarn
yarn start
```

## 特点

- IndexDB 本地存储
- 导入导出 txt 文件
- pc 端，手机端适配

## 主要技术

- typescript
- react
- antd-design
- localForage
